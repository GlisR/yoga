# Документ требований

## Введение

Веб-сайт для преподавателя йоги, предназначенный для предоставления информации о занятиях, расписании, местоположении центра йоги и дополнительных ресурсах (книги, музыка, видео). Сайт должен быть выполнен в теплых, светлых и нежных тонах с использованием только HTML и CSS (W3.CSS фреймворк). Код должен быть простым и понятным для начинающих веб-разработчиков. Сайт должен быть на ангдийском языке

## Глоссарий

- **Сайт йоги** (Yoga Website): Веб-приложение для представления услуг преподавателя йоги
- **Страница** (Page): Отдельный HTML-файл с уникальным содержанием
- **Навигация** (Navigation): Система переходов между страницами сайта
- **W3.CSS**: CSS фреймворк для стилизации веб-страниц
- **Встроенное видео** (Embedded Video): Видео с YouTube, интегрированное на страницу
- **Карта Google** (Google Map): Интерактивная карта с местоположением центра йоги

## Требования

### Требование 1

**Пользовательская история:** Как посетитель сайта, я хочу видеть главную страницу с информацией о преподавателе и йоге, чтобы понять, подходит ли мне этот центр. 

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать главную страницу (index.html) с приветственной информацией
2. Сайт йоги ДОЛЖЕН использовать теплую, светлую и нежную цветовую палитру
3. Сайт йоги ДОЛЖЕН использовать W3.CSS фреймворк для стилизации
4. Сайт йоги ДОЛЖЕН отображать фотографии, связанные с йогой
5. Сайт йоги ДОЛЖЕН содержать текстовый контент о преподавателе и философии йоги
6. Сафйт йоги должен содержать контактную информацию инстаграм страницы и когтактный телефон. Знаки месенжеров для быстрой связи телеграмм, вацап фейсбук
7. Сайт йоги  должен иметь 2 языка переключения английский  и украинский 

### Требование 2

**Пользовательская история:** Как посетитель сайта, я хочу легко перемещаться между разными разделами сайта, чтобы быстро найти нужную информацию.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать ровно 5 HTML-страниц
2. КОГДА пользователь находится на любой странице, Сайт йоги ДОЛЖЕН отображать навигационное меню со ссылками на все 5 страниц
3. Навигация ДОЛЖНА быть видимой и доступной на каждой странице
4. КОГДА пользователь нажимает на ссылку навигации, Сайт йоги ДОЛЖЕН перейти на соответствующую страницу
5. Навигация ДОЛЖНА быть стилизована с использованием W3.CSS

### Требование 3

**Пользовательская история:** Как посетитель сайта, я хочу просматривать рекомендованные книги по йоге, чтобы углубить свои знания.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать отдельную страницу с разделом книг
2. Страница книг ДОЛЖНА отображать список или коллекцию книг по йоге
3. Страница книг ДОЛЖНА включать названия книг и описания
4. Страница книг ДОЛЖНА содержать изображения обложек или тематические фотографии


### Требование 4

**Пользовательская история:** Как посетитель сайта, я хочу слушать музыку для медитации или йоги, чтобы создать подходящую атмосферу для практики.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать отдельную страницу с музыкой или плейлистом
2. Страница музыки ДОЛЖНА включать аудиоплеер HTML5 для воспроизведения музыкальных файлов
3. Страница музыки ДОЛЖНА отображать список доступных треков
4. КОГДА пользователь нажимает на трек, аудиоплеер ДОЛЖЕН воспроизводить выбранную музыку

### Требование 5

**Пользовательская история:** Как посетитель сайта, я хочу смотреть видео с тренировками по йоге, чтобы практиковать дома.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать отдельную страницу с видео тренировками
2. Страница видео ДОЛЖНА включать встроенные видео с YouTube
3. Страница видео ДОЛЖНА отображать минимум 2-3 видео тренировки
4. КОГДА пользователь нажимает на видео, встроенный плеер YouTube ДОЛЖЕН воспроизводить видео

### Требование 6

**Пользовательская история:** Как посетитель сайта, я хочу видеть расписание занятий, чтобы планировать свое посещение центра йоги.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать отдельную страницу с расписанием занятий
2. Расписание ДОЛЖНО быть представлено в виде HTML-таблицы
3. Таблица расписания ДОЛЖНА включать дни недели и время занятий
4. Таблица расписания ДОЛЖНА указывать типы занятий или уровни сложности
5. Таблица ДОЛЖНА быть стилизована с использованием W3.CSS для читаемости

### Требование 7

**Пользовательская история:** Как посетитель сайта, я хочу видеть местоположение центра йоги на карте, чтобы легко найти дорогу.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН содержать страницу с контактной информацией
2. Страница контактов ДОЛЖНА включать встроенную карту Google Maps
3. Карта ДОЛЖНА отображать точное местоположение центра йоги
4. Страница контактов ДОЛЖНА содержать адрес центра йоги в текстовом формате
5. Страница контактов ДОЛЖНА включать номер телефона и месенжеры телеграмм, вацап и инстаграм  и емейл для связи

### Требование 8

**Пользовательская история:** Как начинающий веб-разработчик, я хочу, чтобы код был простым и понятным, чтобы я мог его изучать и модифицировать.

#### Критерии приемки

1. Сайт йоги ДОЛЖЕН использовать только HTML и CSS (без JavaScript)
2. HTML-код ДОЛЖЕН быть структурирован с понятными комментариями
3. CSS-стилизация ДОЛЖНА использовать W3.CSS фреймворк
4. Код ДОЛЖЕН быть написан на уровне студента первого курса
5. Структура файлов ДОЛЖНА быть простой и логичной
6. КОГДА разработчик просматривает код, структура и именование ДОЛЖНЫ быть интуитивно понятными
